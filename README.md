# Autotest_And_SQL
# README к автотесту в test_scooter_order.py
# ОПИСАНИЕ: 
- Оптимизированный код автотеста, который проверяет ожидаемый 200 Ok ответ от сервера Яндекс Самоката при выполнении GET-запроса на получение заказа по его трэк-номеру: автотест в документе test_scooter_order.py, конфигурационная информация к тесту лежит в документе data.py
- Используемые источники для автоматизации: Хабр -> материалы про документацию pytest, фикстуры pytest; книга "Укус питона" К. Сварупа в переводя В. Смоляра
# УСЛОВИЯ ЗАПУСКА АВТОТЕСТА:
- Для корректного функционирования теста должны быть установлены пакеты requests и pytest в среде venv (pip install requests / pip3 install requests из под venv; pip install pytest / pip3 install pytest из под venv + указаны корректные системные переменные в Path) 
- Запуск теста выполняется командами pytest -v и pytest test_scooter_order.py в терминале, также автоматически подхватывается во вкладке Testing и в самой консоли пайтестом (см. приложенные png)
